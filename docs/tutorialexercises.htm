WRF-Chem tutorial exercises

These exercises accompany the WRF-Chem tutorials and the WRF-Chem Users Guide. They are designed for WRF-Chem Version 4, but many concepts apply to earlier versions as well.

Use the links below to open the exercise descriptions and download the corresponding input data.

Exercise 1

Basic WRF-Chem model configuration, namelists, and setup.

Exercise 2

Emissions processing with Prep-Chem-Sources.

Exercise 3

Anthropogenic emissions and input fields.

Exercise 4

Compiling and running WRF-Chem with chemistry.
